## Lab 4 – Data Preprocessing (Titanic Dataset)
### Description

This lab focuses on preprocessing the Titanic dataset, including cleaning, handling missing values, treating outliers, normalization, and evaluating dimensionality reduction.

### Key Steps

- Removed irrelevant features (PassengerId, Name, Ticket, Cabin)

- Handled missing values (Age → median, Fare → dropped)

- Applied outlier capping on Fare (10–90%)

- Performed normalization (Z-score and Min-Max)

- Checked correlation and did not apply PCA
